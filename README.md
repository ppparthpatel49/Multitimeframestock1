# TDMB Premium Momentum System

A professional-grade trading system for NSE India.

## Features
- **3/3 Scoring:** MTF-RSI + Absolute Relative Strength + Volume.
- **Portfolio Manager:** Tracks live trades and journals them upon exit.
- **Official Data:** Fetches Nifty 500 directly from NSE archives.
- **Automation:** Fully automated via GitHub Actions.

## Setup
1. Upload all files to root.
2. Set GitHub Action permissions to 'Read and Write'.
3. Add `TELEGRAM_BOT_TOKEN` and `TELEGRAM_CHAT_ID` to Secrets.
